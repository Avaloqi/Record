### 设备参数
在标准的 Linux 环境中，主要通过读取 /proc 目录下的伪文件来获取实时系统信息。  
#### 内存CPU等基本信息
**查看详细内存参数**  
最标准方法  
```
cat /proc/meminfo
```
|参数	|含义	|备注  |
|------|------|------|
|MemTotal	|系统安装的物理内存总量（RAM）| 	
|MemFree	|当前未被使用的空闲内存量|	  
|MemAvailable	|真正可被应用程序使用的内存量|	考虑到内核缓存和缓冲区后，比 MemFree 更准确|
|Buffers	|用作缓冲区（buffer）的内存量|	用于块设备 I/O，如磁盘读写|  
|Cached	|用作页面缓存（cache）的内存量|	用于文件系统 I/O，存储最近访问的文件内容|
|SwapTotal	|交换空间（Swap）总量|	如果为 0，表示没有使用交换空间|
|SwapFree	|空闲的交换空间量|

free 命令是一个更简洁、更直观的内存和交换空间使用情况摘要工具
```
free -h
```

**查看 CPU 和系统信息**
```
cat /proc/cpuinfo
```

**🔍 查看设备基础信息参数的常用命令**  
系统和内核信息 (高重要性/高常用度)
|命令 |作用 |示例输出及分析 |
|------|------|------|
|uname -a |查看内核版本和系统架构。这是了解系统核心运行环境的关键。|Linux (none) 4.9.118 #0 Tue Apr 12 10:30:52 CST 2022 armv7l GNU/Linux 分析： 运行 Linux，内核版本 4.9.118，编译于 2022 年，架构是 armv7l |
|lsb_release -a |查看发行版信息。（如果安装了 LSB）|Distributor ID: Debian Release: 9.13 分析： 快速了解当前系统的发行版类型（如 Debian, Ubuntu, OpenWrt 等）|
|dmesg |查看内核消息缓冲区。包含系统启动、硬件识别、驱动加载和错误信息|启动后立即运行，可以排查设备初始化、驱动是否正常加载等问题|


进程和负载信息 (高重要性/高常用度)
|命令|作用|示例输出及分析|
|------|------|------|
|top |实时查看系统状态和进程活动。可以动态监控 CPU、内存和各个进程的使用情况。（按m按照内存排序）|Tasks: 120 total, 1 running, 119 sleeping... Cpu(s): 0.5% us, 1.5% sy, 0.0% ni, 98.0% id... 分析： id (空闲百分比) 是衡量设备负载的核心指标。|
|uptime |查看系统运行时间和平均负载。|10:30:00 up 2 days, 15:20, 1 user, load average: 0.10, 0.25, 0.40 分析： load average 三个值分别代表过去 1 分钟、5 分钟、15 分钟的平均负载。对于单核设备 (您的设备)，平均负载持续大于 1.00 就意味着系统负载过重。|
|ps aux |查看所有正在运行的进程。常用于排查哪个进程占用了 CPU 或内存。|配合 grep 查找特定进程，如 `ps aux|

存储和文件系统信息 (中等常用度)
|命令|作用|示例输出及分析|
|------|------|------|
|fdisk -l|查看磁盘分区信息。需要 root 权限。|显示 /dev/mmcblk0、/dev/ubi 等设备的物理分区布局和大小。|
|lsblk|以树状结构列出所有块设备。更现代、更易读的分区结构视图。|确认 SD 卡、MMC 或闪存设备是否被正确识别和分区。|

网络接口信息 (高重要性/高常用度)
|命令|作用|示例输出及分析|
|------|------|------|
|ip addr 或 ifconfig|查看网络接口状态。显示 IP 地址、MAC 地址、接口状态 (UP/DOWN)。|确认设备是否正确获取到 IP 地址，以及网络接口名称（如 eth0, wlan0）|
|ip route 或 route|查看路由表。显示默认网关、子网掩码等信息|确认设备能否正确访问外部网络|
|ping|测试网络连通性。|用于测试设备到网关或云服务器的连通性。|

硬件信息 (中等常用度)
|命令|作用|示例输出及分析|
|------|------|------|
|cat /proc/interrupts|查看中断使用情况。|用于调试驱动和性能问题，看哪些设备或 CPU 占用了大量中断|
|cat /proc/iomem|查看物理内存的 I/O 映射。|显示硬件（如显卡、网卡）占用的内存地址范围|
|cat /proc/cmdline|查看内核启动参数。|包含系统启动时传递给内核的配置信息（如根文件系统位置）|
